# Notes
~~~
Options:
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -b, --base     Base number to be multiply from 1 to 10     [number] [required]
  -l, --list     Show the operations                  [boolean] [default: false]
  -t, --to       Set the limit of the multiplications     [number] [default: 10]
~~~

## Usage
~~~
node app -b=10 -l -t=20
~~~
